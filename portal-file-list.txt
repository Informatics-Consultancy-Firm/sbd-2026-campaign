ICF-SL PORTAL — FOLDER PER MODULE
========================================

Each module has its OWN folder next to icf-sl-portal.html.
Inside every folder, use these EXACT 4 file names:
   presentation.pptx      (sub-module 1)
   video.mp4              (sub-module 2)
   assessment.html        (sub-module 3)
   data-collection.html   (sub-module 4)
